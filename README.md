# mackerel-errorbudget-calculator

## Usage

```
Usage:
  mackerel-errorbudget-calculator [OPTIONS]

Application Options:
  -o, --org=
  -s, --service=
  -r, --roles=
  -f, --filter=
  -m, --metrics=
      --objective-latency= Specify latency in the range of float64 (default: 100)
      --errorbudget-size=  Specify SLO value 0.0 ~ 100.0 (default: 99.9)
      --timewindow=        Specify time window (unit: day) (default: 30)
      --prefix=
      --version            print version and exit
      --debug              Enable debug mode

Help Options:
  -h, --help               Show this help message
```
